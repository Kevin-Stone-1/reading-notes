# What is Markdown?
Markdown is a wonderfully simple way to write easy syntax for formatting plain text on GitHub.

### Three Basic Markdown Formats
1. Heading: by using the \# you can create a heading 1-6
   - # Heading 1 
       -\# Heading 1
   - ## Heading 2
       -\## Heading 2
   - ### Heading 3
       -\### Heading 3
   - #### Heading 4
       -\#### Heading 4
   - ##### Heading 5
       -\##### Heading 5
   - ###### Heading 6
       -\###### Heading 6
